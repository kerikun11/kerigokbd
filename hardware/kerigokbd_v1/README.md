## スペック

| 項目         | 値                         | 注記                         |
| :----------- | :------------------------- | :--------------------------- |
| マイコン     | RP2040                     | Dual Cortex-M0+ 133MHz 264kB |
| フラッシュ   | W25Q16JVUXIQ               | 2MB                          |
| キースイッチ | Kailh Deep Sea Silent Mini | Low Profile                  |
| キー数       | 48 (24 + 24)               | 左右対称                     |
